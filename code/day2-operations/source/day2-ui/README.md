# Day2 UI

## How to Run the Service 

1. Install NodeJs Modules
    
   ```shell
   $ npm install
   ```
   
2. Start the Service
 
   ```shell
   $ npm run serve
   ```
   
   As result the server should show you in the comand window on which port it is running. 
3. Open accoring url in Browser. Eg if running at 8080
   
   ```
   http://localhost:8080/  
   ```
